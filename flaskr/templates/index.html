Hellor World!

Esse é projeto está sendo desenvolvido como parte da disciplina de Programação Orientada a Objetos. Com o objetivo de construir um sistema educaional de quizzes, demonstrando plenitude no que tange ao uso de classes, APIs com o mini-framework flask, persistência de dados com Sqlite3 e templates com HTML, CSS e JavaScript.

Primeiramente, obrigado por acessa o projeto! Sua construção está sendo realmente muito desafiadora, mas também divertida - relembrando e aprendendo novos conceitos das stacks selecionadas. Você pode começar a utilizar as funcionalidades pela barra de navegação acima, registrando e respondendo quizzes, validando o histórico de respostas com statísticas, fazendo login e loggout.
